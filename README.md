# Proyecto Final

## Instalación

```shell
npm install
```

## Ejecutar el proyecto

```shell
npm run dev
```

## Documentación de la API

### Obtener todos los productos

- **GET** `/products`

### Obtener producto por ID

- **GET** `/products/:id`

### Crear un producto

- **POST** `/products`

### Actualizar un producto

- **PUT** `/products/:id`

### Eliminar un producto

- **DELETE** `/products/:id`
